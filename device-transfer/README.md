# Device Transfer

PCとスマホの間で、メッセージ、URL、写真、ファイルを受け渡しするローカルアプリです。

## 使い方

1. `start-device-transfer.bat` をダブルクリックします。
2. 画面に表示される `Phone:` のURLをスマホで開きます。
3. PCとスマホの両方で同じ画面を開いたまま、メッセージやファイルを送ります。

## 注意

- PCとスマホは同じWi-Fiに接続してください。
- PC側の起動画面を閉じると使えなくなります。
- Windowsファイアウォールが表示された場合は、プライベートネットワークで許可してください。
- 受け渡ししたファイルは `storage/uploads` に保存されます。
- これは自分のPCとスマホ用のローカル受け渡しアプリです。インターネット公開用ではありません。
